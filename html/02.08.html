HTML 학습 가이드

HTML에서 제목을 만들 때는 <h1> 태그를 사용합니다.

기본 HTML 구조

<!DOCTYPE html> 
<html lang="ko">
<head>
 <meta charset="UTF-8">
 <title>제목</title>
</head>
<body>
 <h1>안녕하세요!</h1>
</body>
</html>

좋은 코드는 읽기 쉬운 코드입니다.

JavaScript 변수 선언

변수란?

변수는 데이터를 저장하는 공간입니다. JavaScript에서는 var, let, const 키워드를 사용합니다.

기본 사용법

// 변수 선언과 할당
let userName = "김개발";
const age = 20;
var isStudent = true;

// 변수 사용
console.log("이름: " + userName);
console.log("나이: " + age);

전문가 조언

현대 JavaScript에서는 let과 const를 사용하고, var는 피하는 것이 좋습니다. 이는 스코프 문제를 방지하고 더 안전한 코드를 작성할 수 있게 해줍니다.
